# Agency Logo Branding - Quick Start Guide

## For Administrators

### How to Upload Your Agency Logo

1. **Navigate to Admin Settings**
   - Click the settings icon in the user menu
   - Or navigate to: `/admin/settings`

2. **Go to the Branding Tab**
   - Look for the "Branding" tab with the palette icon
   - Click to open the branding section

3. **Upload Your Logo**
   - **Option A - Drag and Drop:**
     - Drag your logo file directly into the upload area
   - **Option B - File Picker:**
     - Click the upload area to open file dialog
     - Select your logo file

4. **Review the Preview**
   - Your logo appears in the preview area
   - Make sure it looks correct
   - Shows how it will display in navigation

5. **Upload**
   - Click the "Upload Logo" button
   - Watch the progress bar
   - Wait for success notification

6. **Done!**
   - Your logo now appears in:
     - Navbar (top bar on all pages)
     - Sidebar header (left sidebar on desktop, drawer header on mobile)

### Requirements
- **Maximum file size:** 5MB
- **Recommended dimensions:** 200x60px to 400x120px
- **Format:** Square or rectangular logos work best
- **Transparency:** PNG and SVG support transparency

### Troubleshooting

**Logo doesn't appear after upload:**
- Refresh your browser (F5)
- Clear browser cache (Ctrl+Shift+Delete)
- Log out and back in
- Check file was actually uploaded

**Upload fails:**
- Check file size (must be under 5MB)
- Verify file format is supported
- Try a different file
- Contact IT support if problem persists

**Logo looks bad in the navbar:**
- Logo might be too small - try higher resolution
- Logo might have poor contrast - adjust colors
- Try converting to PNG format
- Test with different image dimensions

## Technical Details

### What Happens When You Upload

1. File is validated (type, size)
2. File is saved to server: `uploads/logos/filename.png`
3. Logo URL is stored in your user account
4. All pages you visit use your logo
5. Logo persists across sessions

### What if No Logo is Uploaded

- A professional building icon appears instead
- Styled to match the app theme
- Same size and placement as logos
- Still clickable for dashboard navigation

### Logo Storage

- Files stored server-side
- URL stored in database
- Persistent across logins
- Dedicated uploads folder with automatic naming
